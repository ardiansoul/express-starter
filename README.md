# Express-starter

Express Starter write for fasting my development server

## Dependencies

* Express
* Sequelize
* JsonWebToken
* Bcrypt
* Pg
* Multer
* Body-parser
* Sequelize-cli

## Dev Dependencies

* Nodemon
* Dotenv

## Install

1. npm install
2. npx sequelize db:migrate (if you have setup the database in sequelize)
3. npm run dev(for development) or npm run start(for production)
